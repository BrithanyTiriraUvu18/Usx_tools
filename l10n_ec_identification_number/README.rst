============

To install this module, you need to:

#. Copy module files to addon folder.
#. Restart odoo service (sudo service odoo-server restart).
#. Go to your odoo instance and open apps (make sure to activate debug mode).
#. click on update app list.
#. search module name and hit install button.

To use this module, you need to:

#. Go to general settings in electronic documents
#. To autocomplete the data, check and save.
#. Create a new contact
#. If you have the RUC of the person, whether natural or legal, enter it and the fields will be filled automatically, otherwise you must manually fill in each of the fields.
#. Save
