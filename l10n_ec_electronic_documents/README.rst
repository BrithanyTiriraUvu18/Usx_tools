Installation
============

To install this module, you need to:

#. Copy module files to addon folder.
#. Restart odoo service (sudo service odoo-server restart).
#. Go to your odoo instance and open apps (make sure to activate debug mode).
#. click on update app list.
#. search module name and hit install button.

Configuration
=============

[ This file is optional, it should explain how to configure
  the module before using it; it is aimed at advanced users. ]

To configure this module, you need to:
#. Install this module
#. Go to General settings.
#. Go to the selection of electronic documents
#. Configure defalut values
#. Go to the companies section, go to manage company
#. Fill in the general information fields
#. Go to the electronic documents section and fill out the fields
#. Save and continue with the bi_journal_base_sec configuration.

#. Go to invoicing and accounting
#. Go to configuration and select journals
#. Configure customer invoice and vendor invoices
#. Configure the sequence settings section

Usage
=====

To use this module, you need to:

#. Select accounting
#. Select customer invoice
#. Select a Customer
#. Fill in the fields respectively
#. Select a product and confirm
#. Send to the SRI using the Send e-Document button (if the sending was successful, a message will appear and the button will disappear, otherwise the button will change color and will be added to the attempts to send to the SRI).
#. To send the invoices in batches we go to the electronic documents section
#. Select electronic invoice
#. Mark all the invoices in the action button
#. Press Send E-Lot and it will show us a message of correct sending, otherwise it will show us an error message.
