Installation
============

To install this module, you need to:

#. Copy module files to addon folder.
#. Restart odoo service (sudo service odoo-server restart).
#. Go to your odoo instance and open apps (make sure to activate debug mode).
#. click on update app list.
#. search module name and hit install button.

Configuration
=============

[ This file is optional, it should explain how to configure

To configure this module, you need to:

#. Install this module
#. Go to General settings.
#. Go to the selection of invoicing
#. Select the fiscal localization part of the Fenix ERP chart of accounts package

#. Accounting Module
#. SRI tab
